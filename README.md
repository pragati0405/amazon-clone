 # 🛒 Amazon Clone

A responsive frontend clone of the Amazon e-commerce website, built to practice UI/UX design, layout structuring, and frontend development skills. 

## 🔗 Links
* **🌍 Live Demo:** [Click here to view the live website](https://pragati0405.github.io/amazon-clone/)
* **💻 Source Code:** [View the repository](https://github.com/pragati0405/amazon-clone)

## 🚀 Features
* Clean and user-friendly interface inspired by Amazon.
* Fully responsive design for various screen sizes.
* Structured navigation bar with search input.
* Organized product grids and sections.
* Hover effects and interactive UI elements.

## 🛠️ Tech Stack
* **HTML5:** For structuring the web pages.
* **CSS3:** For styling, Flexbox .
 

## 💡 What I Learned
Building this clone helped me strengthen my frontend fundamentals, especially in breaking down complex layouts, mastering CSS Flexbox, and writing clean, maintainable code.
